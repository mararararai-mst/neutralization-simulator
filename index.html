⚗️ 中和シミュレーション

中3 理科 ／ 酸・アルカリと中和（H⁺ + OH⁻ → H₂O）

📘 中和とは何が起きているか

酸の H⁺（水素イオン）と、アルカリの OH⁻（水酸化物イオン）が出会うと、結合して 水（H₂O） になります。

この反応のことを 中和といいます。

H⁺ + OH⁻ → H₂O

シミュ画面では、H⁺ と OH⁻ が近づくと 光って水になって消えるアニメで表現しています。

もっと詳しく：塩（えん）って？

H⁺ と OH⁻ は水になって消えますが、Cl⁻ と Na⁺ はそのまま残っています。

HCl + NaOH の中和でできた水溶液を蒸発させると、塩化ナトリウム（NaCl）の結晶が出てきます。これを 塩（えん） といいます。

反応式: HCl + NaOH → NaCl + H₂O

📍 中和では水＋塩ができる。塩の正体はCl⁻とNa⁺のペア。
シミュの注意点（電離について）

実際の HCl や NaOH は、水に溶けた時点で すべて電離して イオンになっています。

このシミュレーションでは「1滴ずつ追える」よう、滴下の瞬間に電離する演出にしてあります。

📊 4本の棒グラフの読み方

反応するイオン（H⁺ と OH⁻）— ぶつかって水になって消えるので、両方とも数が減っていく。

反応しないイオン（Cl⁻ と Na⁺）— 滴下した分だけ増えて、減らない。最後まで残る。

HClビーカー + NaOHを加える場合

スタート: H⁺ ×5、Cl⁻ ×5 （酸性・黄色）

1滴ごと: Na⁺ +1、OH⁻ +1 がビーカーに加わる
→ OH⁻ は H⁺ と出会って水になり、H⁺と OH⁻ がペアで減る
→ Cl⁻ と Na⁺ はそのまま増えていく

中性: H⁺ = 0、OH⁻ = 0、Cl⁻ = Na⁺ = 5（緑色）

加え過ぎ: もう中和の相手（H⁺）がいないので、OH⁻ が余って増えていく（青色）

📍 プリント①の⑤〜⑧で書くグラフはこの動き。
NaOHビーカー + HClを加える場合

スタート: Na⁺ ×5、OH⁻ ×5 （アルカリ性・青色）

1滴ごと: H⁺ +1、Cl⁻ +1 が加わる

OH⁻ と H⁺ がペアで減り、Cl⁻ と Na⁺ がそのまま増える。

中性を越えると、相手のいない H⁺ が余って酸性（黄色）になる。
